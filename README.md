# MorganLife

A tiny browser prototype inspired by a Nintendo-ish life sim.

## Prototype

- Town square intro
- Pick 1 of 3 houses
- WASD / mouse / touch movement
- NPCs that wander and can speak a simple offline line
- Enter/leave a house
- Basic house interior with couch and kitchen

## Notes

This is intentionally small and browser-only for mobile-friendly testing.

## Run

Open `index.html` in a browser or host the folder as static files.
